# Dekamara Client

`kirka.io` 向けの多機能カスタムクライアントです。
パフォーマンスの最適化、CSS/UserScriptのサポート、便利なショートカットキーなどを提供します。

## ✨ 主な機能 (Features)

* **Unlimited FPS:** FPSの上限を撤廃します
* **CSSローダー:** 独自のCSSをロードして見た目を変更できます
* **UserScriptローダー:** 独自のUserScript（`.js`）をロードして機能を追加できます
* **パフォーマンスの最適化:** よりスムーズなゲームプレイを実現します
* **リソーススワッパー:** ゲーム内のリソース（テクスチャやサウンドなど）を置き換えます
* **便利なショートカットキー:** 下記の「使い方」を参照
* その他、多くの設定オプション

## 💻 動作環境 (Requirements)

* **OS:** Windows
* **依存関係:** 特になし（インストーラーに必要なものはすべて含まれています）

## 🚀 導入方法 (Installation)

1.  [リリースページ](https://github.com/YOUR_USERNAME/YOUR_REPO/releases) から最新版の `Dekamara-Client-Setup-x.x.x.exe` をダウンロードします。
2.  ダウンロードしたインストーラーを実行し、画面の指示に従ってインストールします。

## 使い方 (Usage)

クライアントを起動すると、kirka.io が表示されます。
以下のショートカットキーが利用可能です。

| キー | 機能 |
| :--- | :--- |
| `F1` | 設定メニューを開く |
| `F5` | クライアント（ページ）をリロード |
| `F6` | クイックジョイン（新しいゲームに素早く参加） |
| `F7` | 現在のゲームURLをクリップボードにコピー |
| `F11` | フルスクリーンの切り替え |
| `F12` | 開発者ツールを開く |
