* **Unlimited FPS:** Uncaps your framerate for smoother gameplay.

## How to Use (Usage)

When you launch the client, kirka.io will load.
The following hotkeys are available:

| Key | Function |
| :--- | :--- |
| `F1` | Open Settings Menu |
| `F5` | Reload Client (Page) |
| `F6` | Quick Join (Join a new game) |
| `F7` | Copy current game URL to clipboard |
| `F11` | Toggle Fullscreen |
